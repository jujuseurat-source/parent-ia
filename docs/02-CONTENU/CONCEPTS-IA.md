Roadmap des concepts et activités futures.

## 📚 Pipeline : comment sélectionner un concept

1. **Penser à une activité réelle** : Qu'a fait Romane/Meryl récemment ?
2. **Identifier le concept IA** : Quoi appelle-t-on ça en IA ?
3. **Tester l'activité** : Refaire avec les enfants, noter réactions
4. **Rédiger** : Suivre la structure articles (voir `REGLES-ECRITURE.md`)
5. **Publier** : Copier dans `content/articles/mon-slug.mdx`

## 🎯 Concepts prioritaires (ordre suggéré)

### Phase 1 : Fondamentaux (mois 1-2)

#### 1️⃣ **Classification** ✅ FAIT
- **Article** : "Le tri magique — classer comme une IA"
- **Activité** : Trier jetons par couleur, taille, etc.
- **Concept** : Comment reconnaître des catégories
- **Âge** : 6-9 ans
- **Durée** : 20 min

#### 2️⃣ **Mémoire & apprentissage**
- **Activité** : Jeu de mémoire avec cartes, patterns
- **Concept** : Comment on retient, reconnaît
- **Piste** : Romane vs Meryl — différents styles de mémoire
- **Âge** : 6-8 ans
- **Durée** : 25 min

#### 3️⃣ **Pattern recognition**
- **Activité** : Compléter une suite (couleurs, formes, nombres)
- **Concept** : Voir une règle cachée, prédire la suite
- **Piste** : Dessins, blocs, dominos
- **Âge** : 5-8 ans
- **Durée** : 20 min

#### 4️⃣ **Logique & règles**
- **Activité** : Jeu des questions pour deviner objet
- **Concept** : Poser des questions logiques pour éliminer possibilités
- **Piste** : "C'est animal ? C'est vivant ?"
- **Âge** : 6-9 ans
- **Durée** : 20 min

### Phase 2 : Avancé (mois 3-4)

#### 5️⃣ **Langage naturel**
- **Activité** : Interpréter des instructions floues
- **Concept** : Comment l'IA comprend ce qu'on demande
- **Piste** : "Construis quelque chose" → pourquoi c'est ambigu
- **Âge** : 7-9 ans
- **Durée** : 30 min

#### 6️⃣ **Reconnaissance visuelle**
- **Activité** : Décrire une image, reconnaître objets
- **Concept** : Comment encoder visuellement
- **Piste** : Lego bricks, photos avec clues
- **Âge** : 6-9 ans
- **Durée** : 25 min

#### 7️⃣ **Prédiction & probabilités**
- **Activité** : Deviner avant de voir
- **Concept** : Estimer chances, probabilités
- **Piste** : Dés, cartes, tirages
- **Âge** : 7-9 ans
- **Durée** : 30 min

#### 8️⃣ **Créativité & génération**
- **Activité** : Créer histoires suivant une structure
- **Concept** : L'IA peut inventer du nouveau en suivant patterns
- **Piste** : Mad Libs, générateurs de phrases
- **Âge** : 6-9 ans
- **Durée** : 30 min

### Phase 3 : Expérimental (mois 5+)

#### 9️⃣ **Jeux & algorithmes**
- **Activité** : Résoudre un puzzle, trouver chemin optimal
- **Concept** : Algorithmes, optimisation
- **Piste** : Dédales, puzzles logiques
- **Âge** : 7-10 ans
- **Durée** : 40 min

#### 🔟 **Erreur & apprentissage**
- **Activité** : Entraîner quelqu'un (ou un "robot") à faire une tâche
- **Concept** : Les erreurs sont essentielles pour apprendre
- **Piste** : Enseigner à Meryl, noter où il se trompe
- **Âge** : 6-9 ans
- **Durée** : 30 min

#### 1️⃣1️⃣ **Réseau neural simplifié**
- **Activité** : Transmettre un message en chaîne (téléphone arabe)
- **Concept** : Comment l'info se propage, transformation
- **Piste** : Jeu de transmission d'infos
- **Âge** : 7-10 ans
- **Durée** : 25 min

## 🎨 Thèmes transversaux à explorer

- **Différence Romane vs Meryl** : styles d'apprentissage différents
- **Erreur comme feature** : pas comme bug
- **Intuition vs logique** : quand chacun s'active
- **Créativité des enfants** : comment ça vient
- **Limites humaines** : quand on est fatigués, on oublie

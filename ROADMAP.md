**Concept clé :** 1-2 articles/mois, cycle test → rédaction → publication

1. **Brainstorm** : Quelle activité ? Quel concept IA ?
2. **Test avec les enfants** : Romane + Meryl, noter leurs réactions
3. **Rédaction** : Structurer selon les règles (voir `02-CONTENU/REGLES-ECRITURE.md`)

5. **Publish** : Copier dans `content/articles/`, build local, push repo
